# AI-Batch-Edit

万能 AI 批量编辑器，面向素材工作流的 Windows 批量双图生成工具。

## 功能

- 图1、图2双上传，支持文件夹选择、拖拽追加和单张删除。
- 提示词支持历史记录，点击即可加载；已有内容时会自动空格追加。
- 生成前会校验提示词、素材和 API Key。
- 支持比例、尺寸和并发数量配置。
- 生成结果自动保存到本地输出目录。
- 普通版和混淆版都支持构建。

## 目录规则

- 输出文件夹按日期和分钟创建，例如 `2026-06-12/14：02`。
- 同一分钟重复生成会落到同一个文件夹。
- 图片命名为 `0001.png`、`0002.png` 这种格式。
- 如果同名文件已存在，系统会自动追加 `-2`、`-3`，避免覆盖。

## 本地开发

```bash
npm install
npm start
```

## 构建

普通版：

```bash
npm run dist:win
```

混淆版：

```bash
npm run dist:win:protected
```

## 说明

- API Key 会保存在本机用户目录下的配置文件中，不会写进仓库文档。
- 历史提示词和生成记录都保存在本机配置和输出目录中，发布时不会一起打进混淆包。
- 如果你要重新打包发布，先确认普通版和混淆版使用的是同一套当前源码。
