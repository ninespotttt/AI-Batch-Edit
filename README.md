# AI-Batch-Edit

支点引入-万能AI批量编辑器，一个面向引流素材处理的 Windows 轻量级批量双图 AI 编辑应用。

## 功能

- 图1、图2双上传区，支持文件夹、拖拽、追加图片和单图删除。
- 图1 x 图2 全组合批量生成。
- RunningHub API Key 用户自填。
- 支持比例、2K/4K尺寸和并发设置。
- 生成结果自动保存到本地输出目录。

## 开发

```bash
npm install
npm start
```

## 构建 Windows 便携版

```bash
npm run dist:win
```

输出文件位于 `release/支点引入-万能AI批量编辑器-win.zip`。
